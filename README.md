This is a [Next.js](https://nextjs.org) slide deck for the DNS and Tailscale presentation for an ELI5 meeting on March 2025.

## Get started

1. Install dependencies:

    ```shell
    yarn install
    ```

1. Start the development server:

    ```shell
    yarn dev
    ```

1. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
